Niklas Luhmann abordó el sistema político desde la perspectiva de la teoría de sistemas. Para él, la política es un sistema social autónomo que se distingue por su función de generar decisiones colectivamente vinculantes para la sociedad. Este sistema opera mediante el código binario "gobierno/oposición", lo que permite procesar la complejidad social al decidir quién ejerce el poder en un momento dado. Luhmann argumenta que el sistema político no tiene una relación directa de subordinación con otros sistemas, como la economía o el derecho, sino que interactúa con ellos de manera estructuralmente acoplada. Por ejemplo, las políticas económicas no son dictadas exclusivamente por el sistema político, sino que este debe responder a las expectativas y restricciones impuestas por la economía. El sistema político, según Luhmann, depende de la comunicación para reproducirse y mantenerse operativo. Las decisiones políticas no solo resuelven problemas inmediatos, sino que también producen expectativas para el futuro. La legitimidad, entonces, no se basa únicamente en valores o consensos, sino en la capacidad del sistema político para manejar la incertidumbre y garantizar la estabilidad en un entorno complejo. En esencia, Luhmann veía la política no como un campo de lucha de clases, sino como un sistema autorreferencial y funcionalmente diferenciado que busca mantener el orden social mediante la comunicación y la decisión
